Animalese Typing BetterDiscord Plugin

A plugin for BetterDiscord that emulates Animal Crossing character speech based on what you're currently typing!

Language: JavaScript
Status: RELEASED!
Installation: Place file in BetterDiscord plugin folder, located at "AppData\Roaming\BetterDiscord\plugins" by default.
To open the plugin folder, you can just do "User Settings" > "BetterDiscord" > "Plugins" > "Open Plugin Folder"
