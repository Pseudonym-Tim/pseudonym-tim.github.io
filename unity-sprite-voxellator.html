Unity Sprite Voxellator

A tool for the Unity editor that turns sprites into 3D voxel meshes.
Supports greyscale pixel extrusion mapping and boasts various options and features. Inspired by a very similar tool called KenneyShape.
I originally created this tool to generate 3D model representations of weapons on the player HUD for RONG, one of my game projects!

Language: Unity C#
Status: RELEASED!
Installation: Unzip into your Unity assets folder!
